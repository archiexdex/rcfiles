## Installation

Clone files into `$HOME/rcfiles` by default.

```
git clone https://github.com/archiexdex/rcfiles.git ~/rcfiles
./rcfiles/run.sh
```

**Note: you have to modify `RC_DIR` to `<path-to-rcfiles>` in `.zshrc`.**
